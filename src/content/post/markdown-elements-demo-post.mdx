Markdownは、テキストベースのマークアップ言語で、簡単に書式を付けることができます。この言語は、ウェブ上で記事やドキュメントを書くために広く使用されています。

## <a name="Headings"></a>見出し

Markdownで見出しを作成するには、`#` の記号を使用します。`#` の数が多いほど見出しのレベルは下がります。

## 見出し1

見出し1は、`#` 一つで作成します。最も大きなタイトルに使用されることが多いです。

### 見出し3

見出し3は、`###` 三つで作成します。サブセクションのタイトルに適しています。

#### 見出し4

見出し4は、`####` 四つで作成します。さらに詳細なポイントを説明する際に便利です。

##### 見出し5

見出し5は、`#####` 五つで作成します。特定の詳細や特徴を強調するのに使われます。

###### 見出し6

見出し6は、`######` 六つで作成します。最も小さな見出しで、注目を集めるための追加情報に使用されることがあります。

## <a name="Paragraphs"></a>段落

Markdownで段落を作成するには、単に新しい行でテキストを始めます。行の間に空行を入れることで、段落が分かれます。

これは最初の段落です。ここにテキストが続きます。

これは新しい段落です。Markdownでは、このように簡単に段落を分けることができます。

## <a name="Blockquotes"></a>引用

Markdownでテキストを引用するには、`>` 記号を行の始めに置きます。これは引用文を示すのに使われます。

> これは引用の例です。この行は引用ブロック内にあります。

引用は、他の文書や発言を参照する際に役立ちます。

## <a name="Lists"></a>リスト

### 順序付きリスト

順序付きリストを作成するには、数字とピリオドを使用します。

1. 最初の項目
2. 次の項目
3. さらに別の項目

### 番号なしリスト

番号なしリストを作成するには、`-` または `*` を行の始めに置きます。

- リスト項目1
- リスト項目2
  - サブリスト項目1
  - サブリスト項目2
- リスト項目3

## <a name="Horizontal"></a>水平線

水平線を挿入するには、`---` を使用します。これはセクションの分割に役立ちます。

---

これは新しいセクションの始まりを示しています。

## <a name="Table"></a>表

Markdownで表を作成するには、`|` と `-` を使用して列と行を定義します。

| 項目1 | 項目2 | 項目3 |
|-------|-------|-------|
| データ1 | データ2 | データ3 |
| データ4 | データ5 | データ6 |

表はデータを整理して表示するのに便利です。
